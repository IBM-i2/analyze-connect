    results

    Fetches the results of the specified asynchronous query from the specified queries resource.

    Clients can attempt to retrieve results only from an asynchronous query whose state is 'SUCCEEDED'. An attempt to fetch results from a query in any other state must fail.

    The structure of the response is identical to the response from a synchronous query.

    results

    Request
    GET /{queriesResource}/{queryId}/results
    Parameters
    Name In Type Default Notes
    Authorization header

    If the service requires authentication, a string in the form Bearer token, where token was previously returned by the service following a successful authentication attempt.
    Responses
    Status Code Type Description Samples
    200 I2ConnectData

    The GET method completed successfully.
    401 ProblemDetails

    Authentication failed because the token was missing or not valid.
    500

    The connector failed to retrieve the results of the query.
